# Work assignment

## Description

Work assignment making a product page in React.js

## Installation

To clone

`git clone https://github.com/karinlj/workTest.git`

To install

`npm install`

To run:

`npm start`

To run db with json-server:
In different window

`json-server --watch data/db.json --port 9000`

## To build

`npm run build`

Then use the dist folder you get.

## Built With

- React.js
- Scss
- Font Awesome icons
- json-server
